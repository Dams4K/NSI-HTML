Algorithms

What's an Algorithm?

An algorithm is actually a step-by-step procedure. It is a set of rules to follow to accomplish a task or solve a problem.

Long before the emergence of computing, humans were actually already using algorithms.

In computer programming, algorithms are sets of rules that tell the computer how to perform a task. In reality, a computer program is an algorithm that tells the computer what steps to perform and in what order to achieve a specific task. They are written using a programming language.

The different types of algorithms

There is a wide variety of algorithms, classified according to the concepts they use to accomplish a task. Here are the main categories.

  • "Brute force" algorithms consist of testing all possible solutions until the best one is found. A randomized algorithm uses a random number at least once during the computation to find the solution to the problem.

  • A greedy algorithm aims to find an optimal solution for the overall problem.

  • A recursive algorithm allows you to solve the simplest version of a problem and then solve larger and larger versions until you find the solution to the original problem.

  • A dynamic programming algorithm breaks down a complex problem into a collection of simpler subproblems. All these subproblems are solved once, and their solution is stored for future use.

Sorting algorithms

A sorting algorithm allows you to place the elements of a list in a certain order.

There are many sorting algorithms, with their pros and cons. Here are some examples:

  • Linear sorting algorithms find the smallest items in a list, sort them, add them to a new list, and remove them from the original list.

  • The sort by bubble consists in comparing the first two elements of the list, and inverting them if the first is greater than the second.

  • Insertion sort compares each item in the list with previous items until a smaller item is found.

How are algorithms used in computer science?

It is these instructions that allow the computer to perform tasks. Computer programs themselves are algorithms written in programming languages.

Algorithms also play a key role in the functioning of social networks, for example. They are the ones who decide which publications are displayed or which advertisements are offered to the user.

Introduction

When you want to communicate with someone, you have to know how to speak the same language, otherwise it is difficult to understand each other. Well, with a machine or more particularly a computer it is exactly the same thing.

The difficulty is that a computer only understands 1 and 0. So, to simplify our existence and therefore to be able to communicate easily with a computer, programming languages were created and are used.

There are precisely several types of classifications on its languages.

1 - Machine language

The machine language also called binary language, it is with this language that computers work. It consists of using two states (represented by the numbers 0 and 1) to encode information (text, images, etc.).

In machin language, the programmer must enter each command and all the data in binary form ie in the form of a series of bytes. Each byte comprises a succession of 8 bits. Each bit is represented by a 1 or a 0. Here is an example of a machine language command with 2 bytes: 11001010 00010111.

Let's not forget that any information whether text, images, music or video is encoded in binary language, and to mention the size of a file we use certain measures, namely: the kilobyte, the megabyte, the gigabyte or even the terabyte.

2 - Assembly language

Assembly language is a low-level language close to machine language which can be directly interpreted by the computer's microprocessor.

It consists in representing the combinations of bits used in binary language by easy-to-remember symbols which are then transcribed by a software called assembler in machine language, then executed by the computer.

Unlike a high level language (Java, C, C++), there is a correspondence between the assembly code and the machine language, so it is possible to translate the code in both directions without loss of information.

Each microprocessor has its own machine language, and therefore its own assembly language. This is a problem for computer scientists so much so that to remedy it advanced (or high-level) languages have been created so that they can run on all processors, and all platforms (Windows , Linux, Mac OS).

3 - High-level language

The high-level language is a language that accomplishes a lot for a minimum of code and programming effort. There are a lot of them such as: Pascal, Java, C, C++, C#, Visual Basic, Python, PHP, JavaScript, Perl etc...

Unlike assembly language which is very close to machine language, they offer layers that make the material abstract. The assembler does not use advanced structures such as loops (while, for) and conditions (if, switch) whereas the latter are an integral part in advanced languages.

The family of advanced languages can be divided into two main categories:

a) Script language

A script is a sequence of commands intended to be executed in their order of appearance, line by line, instruction by instruction, which can result in a more or less noticeable slowness.

Scripting languages also use variables, control structures (loops, conditions).

Some examples:

JavaScript which is a code for programming interactive user interfaces useful for programming websites, improving the positions and indexing of pages in search engine results.

PHP whose main role is to develop better communication between the web page and the server. Guarantor of the stability of the pages, it is however invisible on the client side unlike JavaScript.

b) Compiled languages (interpreted languages)

A compiled language is a language where all instructions are translated into binary code before being executed by a compiler.

It is decoded and then executed instruction by instruction during the execution of the program, which translates into faster execution.

Some examples:

The C language which is an essential programming language for programming a video game for example or a Web application on Microsoft. It has evolved with the appearance of new languages using its syntax, such as C++ and C#.

Python is a cross-platform language that allows object-oriented programming and is placed under a free license used in mathematics, finance or scientific research…

4 - Markup language

A markup language is based on so-called tags, which are tags with which data (words, text etc.) to give them meaning (this is a paragraph, this is a title, quote etc.) that can be made visual (italics, bold, text color etc.) and that we can label this data in framing them with these tags.

5 - Query language

A query language most often qualifies the languages specific to databases, they are represented in particular by the SQL (Structured Query Language) in French simply allows you to manage a database, for example to query it, insert data or delete others, ask it to bring out data according to criteria that I set for it.

You have now learned the types of computer languages and the usefulness of having created so many.

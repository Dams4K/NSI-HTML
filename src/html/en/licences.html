Software licenses

What is a license?

A software license is a contract by which the holder of the rights to the software authorizes an outside person to make modifications and to share them (for free or not) or to just read it. He grants therefore to the licensee(s) the right to install the software, use it and make a backup copy.

Using unlicensed software is copyright infringement.

Often, the rights holder not only grants the license, but also adds requirements such as prohibitions:

  • To share the purchased software with others
  • To study the software
  • To modify and share it
  • To resell the same software, modified or not

The different types of licenses :

There are different types of licenses to serve different problems, so no one license is better than another:

  • Fixed license

    It is designed to be installed on a particular computer, sometimes using the MAC (Media Access Control) address to restrict use to that computer.

  • Floating license

    The floating license works on a network. As long as at least one license remains available on the network, any computer connected to the network can claim it and use it for as long as needed.

    Example :This allows a company to buy only a certain number of floating licenses, knowing that the number of employees using the software will not exceed the limit. Not all computers will need to be equipped with the license, which saves a significant amount of money.

  • Shareware license

    Shareware licenses offer a temporary right to use the software for free with limitations. After this trial period, the user will have to pay the owner of the license to continue using the software. The user can also not wait for the end of the free period to have access to all the features.

  • Free licenses

    The free licenses cover free software, they give the user the following four rights:

    • Use of the software.
    • Study the software to understand its mechanisms.
    • Modification or incorporation of the software into a derivative version.
    • Redistribution du logiciel et des versions dérivées, y compris sous forme payantes.

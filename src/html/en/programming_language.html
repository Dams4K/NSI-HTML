What's a "Programming Language":

It's a computer language (so a means of communication). It allows a human to write source code, written in the language corresponding to the computer language. Syntax rules must be respected, these change from one language to another.


Different types of programming language1

Advantages and disadvantages of programming languages:

This computer language system has different good and bad points.

Advantages:
  • To obstruct the original mechanism of the machines.
  • No need to know the language of the machine.
  • Allow different techniques, each adapted to its sector of use. But let's leave that part for the diversity of languages.
Drawbacks:
  • An assembler or compiler is needed to transform the source code into a machine-readable form.
  • Depending on what you want to do, handling several languages is sometimes sometimes mandatory.

Introduction

You might be wondering why are there so many programming languages? Why a universal language does not exist? Why is HTML not a programming language?

The objective of this site is to answer certain questions which you ask yourselves. We're going therefore help you to better find your way around, with the help of this site. Whether you are English or French you will be able to use this site, you can change language at any time by clicking on the button "Settings" at the bottom of the navigation bar.
